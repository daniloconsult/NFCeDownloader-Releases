# NFCe Downloader - Releases

Este repositório contém apenas as versões compiladas (binários) do NFCe Downloader.

## 📦 Download

Para baixar a versão mais recente do aplicativo, acesse a seção [**Releases**](https://github.com/daniloconsult/NFCeDownloader-Releases/releases) deste repositório.

### Como instalar:

1. Acesse a página de [Releases](https://github.com/daniloconsult/NFCeDownloader-Releases/releases)
2. Baixe o arquivo `BuscaCupom.exe` da versão mais recente
3. Execute o arquivo baixado
4. Faça login com suas credenciais fornecidas pelo administrador

## 📝 Código Fonte

O código fonte do projeto está disponível no repositório principal:
**[NFCeDownloader-MultiTenant](https://github.com/daniloconsult/NFCeDownloader-MultiTenant)**

## 🆘 Suporte

Para dúvidas ou problemas, entre em contato através do WhatsApp fornecido pelo administrador do sistema.
